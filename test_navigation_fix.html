问题：任务调度页面中"打开监控页面"按钮点击无响应

原因：按钮使用了复杂的条件表达式 onclick="window.f7?.views?.main?.router?.navigate('/monitoring') || Utils.showNotification('请使用监控页面', 'info')"

解决方案：将复杂的内联表达式替换为专用的导航函数调用

🔧 修复详情

修复前（有问题的代码）：

<button onclick="window.f7?.views?.main?.router?.navigate('/monitoring') || Utils.showNotification('请使用监控页面', 'info')"> 打开监控页面 </button>

修复后（新的实现）：

<button onclick="navigateToMonitoring()"> 打开监控页面 </button> // 新增的导航函数 function navigateToMonitoring() { console.log('尝试导航到监控页面'); if (window.f7 && window.f7.views && window.f7.views.main) { try { window.f7.views.main.router.navigate('/monitoring'); Utils.showNotification('正在打开监控页面...', 'success'); } catch (error) { Utils.showNotification('导航失败，请稍后再试', 'error'); } } else { openMonitoring(); // 回退方案 } }
